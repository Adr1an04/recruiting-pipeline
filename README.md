# Recruiting Pipeline

A local-first workspace for organizing career materials, tracking recruiting progress, and keeping job-search context useful over time.

The project is being designed around privacy, user control, and human review. It is intentionally early-stage; integrations and automation will be introduced incrementally and should remain understandable, reversible, and safe to run locally.

## Status

Early research and architecture phase.

## Principles

- Local-first data ownership
- Explicit permissions and least-privilege integrations
- Human approval for consequential actions
- Durable career context rather than one-off generated output

## Development

Initial research, design decisions, and implementation milestones will be added as the project takes shape.
